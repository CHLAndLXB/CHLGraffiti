# CHLGraffiti
涂鸦

在View上面直接涂鸦

使用方法
```CHLPointView * pointView = [[CHLPointView alloc]initWithFrame:CGRectMake(0, 0, self.view.frame.size.width, self.view.frame.size.height)];
pointView.backgroundColor = [UIColor whiteColor];
[self.view addSubview:pointView];```
